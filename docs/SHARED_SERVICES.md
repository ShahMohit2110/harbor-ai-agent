# Harbor AI - Shared Services

**Last Updated:** 2026-03-24

## Overview

Harbor AI is the framework that provides shared intelligence to all Harbor services.

---

## Services Provided

1. **Documentation Validator** - Validates/generates /docs folders

2. **Agent Definitions** - Backend and frontend agents

3. **Workflows** - Execution protocols

4. **Intelligence** - Decision frameworks

**All Harbor services** use Harbor AI for:
- Documentation standards
- Agent-assisted development
- Cross-service intelligence

**Services import:**
- Agent definitions
- Workflow protocols
